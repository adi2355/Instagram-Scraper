About This Project

Purpose

This knowledge base was created for personal educational use, to make it easier to search, browse, and learn from Instagram content that I have been given permission to access.

How It Works

The system downloads videos from specified Instagram accounts, extracts the audio, transcribes it using OpenAI's Whisper model, and indexes the transcripts in a searchable database.

The web interface allows you to search through all transcribed content, filter by account or tags, and view the original videos along with their transcripts.

Ethical Usage

This tool is for personal educational use only and is used with permission from content creators. All content remains the intellectual property of its original creators.
